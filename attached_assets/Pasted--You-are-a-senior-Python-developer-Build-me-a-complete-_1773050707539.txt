PROJECT OVERVIEW
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
AXIS is used by cybercrime police investigators to analyse
digital evidence from court cases — chat transcripts, email
dumps, witness statements, FIR documents, server logs.

Core problem: Online LLMs like ChatGPT send evidence to
foreign servers, violating chain of custody, IT Act and
CrPC evidence handling rules. Cases get thrown out of court.

AXIS runs a powerful LLM 100% offline on the investigator's
local machine. Zero data leaves the device. Ever.

User flow:
Upload file → Parse and extract entities → Instant AI
analysis → Chat with evidence → Download PDF report

Function: parse_evidence(text: str) -> dict

Extract from any plain text using regex:
- Names: two consecutive Title Case words
- Phones: Indian format \b[6-9]\d{9}\b
- Emails: standard email regex
- URLs: http/https, flag if domain not in
  [sbi.co.in, hdfcbank.com, icicibank.com,
  gov.in, nic.in]
- Dates: DD/MM/YYYY and written month formats
- Locations: match list of 20 Indian cities
- Criminal keywords with occurrence count:
  [threat, kill, leak, blackmail, extort, hack,
  fake, fraud, illegal, maal, supari, payment,
  transfer, account, password, OTP, nude, photo,
  video, money, cash, ransom, demand]

IT Act auto-match rules:
  nude/photo/video/leak/blackmail → 66E, 67, 354D IPC
  OTP/password/fake/phishing/bank → 66C, 66D, 420 IPC
  hack/unauthorised/access/server → Sec 43, 66 IT Act
  fraud/transfer/account/money    → 66D, 420 IPC
  threat/kill/ransom/extort       → 506 IPC, 384 IPC

Return structured dict with all entities

    risk_level: Low|Medium|High|Critical,
    risk_score: 0-100,
    crime_type: short string,
    summary: 3 paragraph plain English,
    key_findings: [6 strings],
    recommended_actions: [4 strings],
    applicable_laws: [sections with descriptions]
  }
- Never hallucinate, only use provided data
- If JSON parse fails retry once
- Return dict

Function: chat_with_evidence(message, entities,
                  raw_text, chat_history) -> str
- System prompt: AXIS forensic analyst for Indian
  law enforcement, answer only from evidence,
  cite IT Act and IPC sections, never hallucinate
- Include entities JSON + raw_text (max 3000 chars)
  in system message as context
- Include full chat_history as conversation memory
- Return reply string
- ConnectionError → clear error message
- Timeout → clear error message
